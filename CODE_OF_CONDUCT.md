# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant] as its Code of
Conduct. By participating, you are expected to uphold it.

Please read the full text here:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to the
maintainer by opening an issue at
<https://github.com/iankressin/contract-state-reproduction/issues>. For reports that should
not be public, please note that in the issue and the maintainer will follow up privately.

All complaints will be reviewed and investigated promptly and fairly, and the maintainer is
obligated to respect the privacy and security of the reporter of any incident.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
